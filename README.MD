# WEB103 Project 2 - Amazing Houseplants

Submitted by: **Le Hai Ha Ngo**

About this web app: **A web app to display information about house plants, now powered dynamically by a Render PostgreSQL database with support for real-time search filtering.**

## Notes

One of the challenges was dealing with external image URLs (like Unsplash) which often expire or break. To solve this, we generated custom photorealistic images and served them locally from our Express backend. The project also focused heavily on delivering a premium aesthetic using pure Vanilla CSS.
